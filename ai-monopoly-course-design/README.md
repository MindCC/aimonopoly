# 《AI 创新应用》模拟经营式综合答辩材料包

## 材料清单

| 文件 | 用途 |
|---|---|
| `01-学生版规则说明.md` | 发给学生，说明游戏背景、角色、资源、课堂经营和期末决战规则 |
| `02-教师版经营判定表.md` | 教师课堂使用，用于判定策略收益、课堂回答奖励和负面行为处理 |
| `03-期末大富翁决战流程.md` | 期末答辩当天使用，包含棋盘、流程、评分表和主持话术 |
| `04-机会卡危机卡道具卡.md` | 可打印或放入 PPT / 抽签工具的卡牌库 |
| `05-经营记录表.md` | 每个小组全学期填写，用作过程性评价证据 |

## 推荐使用顺序

1. 第 1 次课：发放学生版规则，完成分组和角色确定。
2. 第 2-8 次课：每 1-2 次课进行一次经营回合，教师使用经营判定表记录资源变化。
3. 课程中后期：要求各组持续填写经营记录表。
4. 期末前：公布期末大富翁决战流程和评分表。
5. 期末当天：使用棋盘流程、机会卡、危机卡和道具卡完成综合答辩。

## 设计原则

- 现金是经营资源，不是简单积分。
- 角色发言是必需环节，避免一个学生包办全部答辩。
- 随机卡牌增加现场感，但不能决定最终成绩。
- 最终评价仍以 AI 项目质量、技术实现、商业验证和答辩表现为主。
- 过程经营记录可作为平时成绩和期末答辩的重要证据。
