# 《AI 创新应用》模拟经营式综合答辩

## 游戏名称

AI 创业大富翁：从 Agent MVP 到商业路演

## 游戏背景

每个学习小组是一家 AI 创业公司。公司需要在一个学期内完成一个可演示、可解释、可验证的 AI 创新应用，并通过课堂经营、项目开发、市场选择和期末路演获得最终成绩。

本活动不是单纯游戏，而是课程项目制学习的一部分。你们的每一次课堂回答、阶段成果、技术选择、运营策略和风险处理，都会影响公司的现金、用户、产品力、技术力、声誉和合规水平。

## 初始资源

每家公司初始资源完全相同。

| 资源 | 初始值 | 含义 |
|---|---:|---|
| 现金 | 10000 元 | 公司可支配资金，用于研发、推广、合作、风控 |
| 用户数 | 0 | 产品获得的潜在或真实用户数量 |
| 产品力 | 1 | 需求匹配、MVP 完整度、用户体验 |
| 技术力 | 1 | Agent、Prompt、Workflow、知识库、工具调用等能力 |
| 声誉 | 1 | 用户口碑、展示效果、团队可信度 |
| 合规 | 1 | 数据安全、隐私保护、版权和伦理意识 |

## 团队角色

每组建议 4 人。人数不足时可以兼任角色，但期末答辩时每个角色职责都必须有人承担。

| 角色 | 主要职责 | 课堂与答辩任务 |
|---|---|---|
| CEO / 产品经理 | 选题、用户痛点、MVP、功能取舍 | 说明产品解决谁的什么问题，为什么值得做 |
| CTO / 技术负责人 | Agent 架构、Prompt、Workflow、知识库、工具调用 | 说明 AI 应用如何工作，如何优化与保证稳定性 |
| CMO / 运营负责人 | 用户增长、反馈、推广、场景验证 | 说明如何找到用户，如何验证需求和效果 |
| CFO / 风控负责人 | 预算、收益、成本、合规、风险 | 说明钱花在哪里，商业模式和风险控制是什么 |

## 课堂经营回合

课程中会设置若干次经营回合。每次经营回合通常包含 5 个步骤：

1. 教师发布本节课经营事件。
2. 小组讨论并选择 1 个运营策略。
3. 对应角色说明选择理由。
4. 小组完成课堂问题、阶段展示或项目任务。
5. 教师根据表现更新公司资源。

## 常见运营策略

| 策略 | 成本 | 成功效果 | 说明 |
|---|---:|---|---|
| 用户访谈 | -500 | 产品力 +1，声誉 +1 | 需要提交真实或合理的访谈问题与结论 |
| 竞品分析 | -800 | 产品力 +1，现金 +1000 | 需要说清至少 2 个竞品和差异化 |
| Prompt 优化 | -800 | 技术力 +1 | 需要说明优化前后的变化 |
| 工作流重构 | -1500 | 技术力 +2，产品力 +1 | 需要展示流程图或关键节点 |
| 接入知识库 | -2000 | 技术力 +2 | 合规低于 2 时可能触发风险 |
| 快速 Demo | -1500 | 产品力 +2 | 需要展示原型、流程或可运行版本 |
| 广告投放 | -2500 | 用户 +50 | 产品力低于 3 时，声誉 -1 |
| 校园试点 | -1000 | 用户 +30，声誉 +1 | 需要说明真实使用场景 |
| 商业合作 | -1500 | 现金 +4000 | 需要讲清合作对象和收益逻辑 |
| 免费试用 | -1000 | 用户 +60 | 现金低于 3000 时不可选 |
| 合规审查 | -1000 | 合规 +2 | 短期无收入，但可抵御后续风险 |
| 版本迭代 | -1500 | 产品力 +1，技术力 +1 | 需要展示前后变化 |

## 课堂问答奖励

课堂回答和阶段展示会转化为经营收益。

| 表现 | 奖励 |
|---|---:|
| 回答基础概念正确 | 现金 +500 |
| 回答能结合本组项目 | 现金 +1000 |
| 回答有案例、有判断、有反思 | 现金 +1500 |
| 现场演示成功 | 产品力 +1 或技术力 +1 |
| 主动指出数据、伦理或安全风险 | 合规 +1 |
| 给其他小组提出有价值建议 | 声誉 +1 |
| 阶段任务迟交 | 现金 -1000 |
| 项目长期无明显进展 | 产品力 -1 |
| 数据、截图、用户反馈造假 | 现金 -3000，声誉 -2，合规 -2 |

## 期末大富翁决战

期末答辩时，各公司带着整个学期累计的资源进入决战。

期末前，每家公司可以用现金购买道具卡：

- 每 3000 元购买 1 张道具卡。
- 每组最多购买 3 张。
- 现金不能直接换成绩，但会影响期末答辩中的资源优势。

期末答辩由棋盘格、卡牌、项目路演和评委提问组成。你们会遇到产品、技术、市场、风险、资本、机会和危机等不同格子。

## 最终成绩来源

成绩主要来自项目质量、技术实现、商业验证、经营决策、角色协作和答辩表现。游戏资源会影响期末表现，但不会替代真实项目质量。

建议每组保留一份经营记录，包括：

- 每次经营回合选择了什么策略。
- 为什么这样选择。
- 资源发生了什么变化。
- 项目有哪些实际进展。
- 哪些决策成功，哪些决策失败。

## 一句话目标

用一个学期，把你们的 AI 创意经营成一家能够路演的 AI 创业公司。
